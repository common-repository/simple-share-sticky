An easy way to add social media share buttons to any site, without the need to setup anything at Facebook, Twitter, Google etc!

== Description ==
Simple Share Sticky is the perfect plugin for any WordPress user that needs an easier way to add social media share buttons. Simple Share Sticky is so simple that you don’t even need to set anything up at Facebook, Twitter, Google+, etc. to start displaying your social shares!

The Simple Share Sticky plugin is perfect for:
Bloggers, writers, journalists, and reporters
Marketers, content managers, and content creators
Website owners, and more!

All you’ll need to do to get the full use out of Simple Share Sticky is:
Select the social networks you wish to enable on your site
Select where you’d like the social shares to appear. On your homepage? Just on your Blog? Everywhere?

And that’s all there is to it! A sticky Social Share tab will appear on the left hand side of your screen for your users to share while on desktop and along the bottom on mobile. Check out the screenshots!

== Installation ==

Plugin Search (Recommended):

	- In /wp-admin/, go to Plugins > Add New
	- - Search for "Simple Share Sticky"
	- Look for "Simple Share Sticky”
	- Click “Install Now”
	- After installation has completed, click “Activate Plugin”

Manual Upload:

	- Download the plugin .zip file from wordpress.org/plugins/simple-share-sticky/
	- In /wp-admin/, go to Plugins > Add New. Click “Upload”
	- Locate the .zip file on your computer. Click “Install Now”
	- After installation has completed, click “Activate Plugin”

Manual FTP:

	- Download the plugin .zip file from wordpress.org/plugins/simple-share-sticky/
	- Extract the plugin folder contained in the .zip file
	- Upload the plugin folder, via FTP, to your /wp-content/plugins/ directory
	- On your Plugins screen, under Simple Update, click “Activate”
